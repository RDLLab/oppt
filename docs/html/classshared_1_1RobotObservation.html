Detailed Description

A class representing an observation in the RockSample POMDP.

This is represented by two boolean flags; one for whether or not any observation was made, and one for whether the rock was seen as good or bad if an observation was, in fact, made.

This class also implements abt::DiscretizedPoint so that the solver can use a simplistic enumerated observation mapping approach (EnumeratedObservationPool) to store the possible observations from each ActionNode.
