Detailed Description

Contains the StatePool class, which stores all of the states that have been encountered by the solver.

The StatePool's primary purpose is to provide a way to mark which states have been affected by changes, and to retrieve the set of affected states.

The StatePool also stores a StateIndex, the main purpose of which is to provide an efficient way to look up states.
