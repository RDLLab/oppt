Detailed Description

An abstract class that contains the data to construct continuous actions.

The data has to be in vector form. The interface is storage and size agnostic. As such the constructiondata is only accessed though the data() function which returns a pointer to the array data of an std::array or std::vector or similar.

The data() pointer only needs to point to the part relevant for the continuous action space. Additional discrete actions may be handled otherwise.

Note, that a chooser for continuous actions is likely to make assumptions about the size of data(). It is expected that there is one value for each dimension.
