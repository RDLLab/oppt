Detailed Description

template<typename T>
class abt::CachedValue< T >

Represents a cached value associated with a specific belief node.

This is done by wrapping a function that takes a pointer to a belief node and returns a value of the value type together with a pointer to the specific belief node this cache is associated with.

This allows the value to be stored, and also recalculated on demand.
