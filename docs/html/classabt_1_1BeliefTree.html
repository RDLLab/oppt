Detailed Description

Represents a belief tree, which is the core data structure representing a policy for the ABT solver.

This class just owns the root belief node; each individual node in the tree is owned by its parent mapping entry.

However, this belief tree also keeps track of an index of nodes, represented by a vector of non-owning pointers to the individual belief nodes. This allows for access by node ID, iteration and serialization.
