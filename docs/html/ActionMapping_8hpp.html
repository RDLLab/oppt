class  abt::ActionMapping
 

Detailed Description

Contains the abstract base class ActionMapping.

In essence, this class defines, for a single belief node, the mapping of actions to (belief, action) edges in the belief tree - each of these is represented by an ActionMappingEntry.

The purpose of this abstract class is to allow different actions to be grouped together in customizable ways.
